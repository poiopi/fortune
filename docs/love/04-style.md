# 恋愛スタイル 導出仕様書

Styleは、性格プリミティブの組み合わせから導かれる、恋愛における反応・行動の傾向を表す。価値観・経験・年齢の影響が大きい概念（Tendency）とは区別する（[00-overview.md](00-overview.md)）。

## 受け入れ基準

1. 最低2つのPrimitiveから導出されること
2. 各Primitiveが意味のある寄与を持つこと（実質的に単一Primitiveへ縮退していないこと）
3. 同じPrimitiveペアを使う他のStyleがある場合、符号パターンまたは比率が実質的に異なり、ベクトルとして別方向を向いていること（3節参照）
4. 導出式が項目の定義を自然に説明できること（名称ではなく定義文とPrimitiveの対応を見る。数式レビューだけでなく意味レビューを行う）
5. Primitiveの意味論を壊さないこと（[01-principles.md](01-principles.md)の原則11）。負の係数は使ってよいが、Primitive自体の意味を反転させてはならない

導出式の係数は仮値。768通り（MBTI16×血液型4×星座12）の全列挙結果を見てから較正する（[00-overview.md](00-overview.md)のNormalizerと同じ方針）。

## 一覧

| 名称 | 定義 | 導出Primitive | 導出式（仮） | 採用理由 |
|---|---|---|---|---|
| 積極性 | 恋愛において自分から行動し、好意を伝えていく傾向 | 行動主導性 + 情動性 | 0.6×行動主導性 + 0.4×情動性 | 行動主導性単体は一般的な行動力に留まる。情動性が加わることで「好意という感情を行動に変える」恋愛特有の意味が生じる |
| 愛情表現 | 好意を言葉や態度で強く表に出す傾向 | 行動主導性 + 情動性 | 0.3×行動主導性 + 0.7×情動性 | 情動性（想いの強さ）が主導し、行動主導性（それを表に出す力）が補う。積極性とは同じペアだが支配的な項が逆転しており、別のベクトルになる |
| 包容力 | 恋愛において相手を受け入れ、関係を安定して育てようとする傾向 | 誠実性 + 情動性 | 0.6×誠実性 + 0.4×情動性 | 誠実性単体は堅実さに留まる。情動性（相手の感情を受け止める力）が加わることで「包み込む」ニュアンスが生じる |
| 独占欲 | 相手を自分だけのものにしたいと感じる傾向 | 情動性 − 自立性 | 0.7×情動性 − 0.3×自立性 | 情動性の強さが独占欲の燃料になり、自立性の低さ（依存傾向）がそれを増幅する |
| 惚れやすさ | 恋愛感情が芽生えるまでの早さ | 情動性 + 変化志向 | 0.6×情動性 + 0.4×変化志向 | 情動性の高さが感情の立ち上がりの早さに関係し、変化志向の高さが新しい出会い・刺激への開放性を後押しする |
| 嫉妬深さ | 相手の言動に不安や嫉妬を感じやすい傾向 | 情動性 − 誠実性 | 0.6×情動性 − 0.4×誠実性 | 情動性の強さが感情の揺れを生み、誠実性（安定・信頼）の低さがその揺れを増幅する |
| 恋愛の慎重さ | 関係を進める際に、勢いに流されず慎重に見極める傾向 | 誠実性 + 自立性 | 0.6×誠実性 + 0.4×自立性 | 誠実性（慎重さ・安定を重んじる）が「見極める」行為を支え、自立性（自分のペースを保つ）が「勢いに流されない」側面を補う |

## 重複確認

| 対象 | 確認内容 |
|---|---|
| 積極性 ⇔愛情表現 | 当初は両方とも「行動主導性+情動性」のほぼ同じ配分で、実質的に同じベクトル（線形独立性の欠如）になっていた。第4基準（名前が導出式を説明できる）に照らすと、ペアを差し替えるより支配的な項を反転させる方が両者の名前と整合する（積極性＝行動主導性優勢、愛情表現＝情動性優勢） |
| 包容力 ⇔ 嫉妬深さ | 同じPrimitiveペア（誠実性+情動性）を使うが、誠実性の符号が逆（包容力は+、嫉妬深さは−）。符号パターンが異なるため別方向のベクトルであり、問題なしと判断 |
| 独占欲 ⇔ 嫉妬深さ | 主軸（情動性）は共通だが、第2 Primitiveが異なる（独占欲は自立性、嫉妬深さは誠実性）。列挙検証時に相関の強さを確認すること |
| 恋愛の慎重さ | 誠実性+自立性は他のどのStyle/Tendencyも使っていない組み合わせ |

## 保留・却下（[05-tendency.md](05-tendency.md)で再評価済み）

浮気耐性はTendencyへ採用、安心感は却下、恋愛の慎重さは上記に採用。詳細な判定理由は05-tendency.mdを参照。
