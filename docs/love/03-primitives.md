# 性格プリミティブ辞書

PrimitiveはShared Axis LibraryのAxisを恋愛ドメインの語彙へ翻訳した最小単位である。Primitive同士は互いに独立し、Style・Tendency・BundleはPrimitiveの組み合わせから導出される。Primitive自体は単一Axisとの1:1対応を維持する。

表示文（結果文・High/Mid/Lowの言い回し等）はここに含めない。Presentation側の文書（Writing Rules／Bundle・Composerのテンプレート）で扱う。

| 名称 | 対応Axis | 使用Trait | 定義（恋愛での意味） |
|---|---|---|---|
| 行動主導性 | ACTION | 行動力（TRAIT_ACTION）／挑戦（TRAIT_CHALLENGE） | 恋愛において自分から動き、関係を前に進める力 |
| 誠実性 | RELIABILITY | 慎重さ（TRAIT_CARE）／責任感（TRAIT_DUTY）／安定性（TRAIT_STABILITY） | 相手との約束や信頼を大切にし、安定した関係を築こうとする力 |
| 情動性 | SENSITIVITY | 直感（TRAIT_INTUITION）／情熱（TRAIT_PASSION） | 相手の感情に共鳴し、自分の感情も豊かに動く力 |
| 自立性 | AUTONOMY | 独立（TRAIT_INDEP） | 恋愛関係の中でも自分の軸・時間を保つ力 |
| 変化志向 | TRANSFORM | 変化（TRAIT_CHANGE） | 恋愛に新しい刺激や変化を求め、楽しむ力 |
